Why Open Source?

This voice interface uses only free, open-source components for speech-to-text and text-to-speech processing:

  • Mozilla DeepSpeech - Open-source speech recognition engine
  • Vosk - Lightweight offline speech recognition toolkit
  • Coqui TTS - Open-source text-to-speech engine
  • eSpeak NG - Compact open-source speech synthesizer

All processing happens locally on your server, with no dependency on commercial cloud APIs.

Try It Out

Click the microphone button and speak to the chatbot. Your voice will be converted to text using open-source speech recognition, processed by the chatbot, and you'll hear the response generated by open-source text-to-speech.

How It Works

Speech-to-Text

Your voice is captured and processed using one of these open-source engines:

  • DeepSpeech Mozilla's neural network based STT engine
  • Vosk Lightweight offline STT toolkit

Both engines run entirely on your server, with no data sent to external services.

Text-to-Speech

The chatbot's response is converted to speech using:

  • Coqui TTS High-quality neural TTS engine
  • eSpeak NG Lightweight speech synthesizer

All voice synthesis happens locally, ensuring privacy and zero usage costs.

How to Implement

Adding open-source voice capabilities to your chatbot is simple. Just include our component and a few lines of code:

// Include the Open Voice Interface component
<script src="/js/components/OpenVoiceInterface.js"></script>

// Add the CSS
<link rel="stylesheet" href="/css/voice-interface.css">

// Create an instance
const openVoiceInterface = new OpenVoiceInterface({
  chatbotId: 'your-chatbot-id',
  showEngineInfo: true,
  onResponse: (data) => {
    // Handle the response
    console.log(data);
  }
});

// Render the component
ReactDOM.render(
  openVoiceInterface,
  document.getElementById('voice-container')
);
