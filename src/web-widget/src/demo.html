Chatbots Platform Web Widget

This is a demo of the embeddable web widget for the Chatbots Platform. Use the controls below to customize the widget and see how it can be integrated into your website.

Quick Start

Add the following script to your website to integrate the chatbot widget:

<script src="https://cdn.chatbots-platform.example.com/widget.js" data-chatbot-widget="true" data-auto-init="true" data-api-key="YOUR_API_KEY" data-chatbot-id="YOUR_CHATBOT_ID" data-position="right" data-theme="light"></script>

Widget Configuration

JavaScript API

You can also initialize and control the widget programmatically:

const widget = new ChatbotWidget({ apiKey: 'YOUR_API_KEY', chatbotId: 'YOUR_CHATBOT_ID', position: 'right', theme: 'light', title: 'Chatbot Assistant', subtitle: 'How can I help you today?', welcomeMessage: 'Hello! How can I assist you today?', primaryColor: '#2563eb', secondaryColor: '#f3f4f6', textColor: '#111827' }); widget.init();
