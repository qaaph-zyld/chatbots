How to Implement

Adding voice capabilities to your chatbot is simple. Just include our component and a few lines of code:

// Include the Voice Interface component
<script src="/js/components/VoiceInterface.js"></script>

// Add the CSS
<link rel="stylesheet" href="/css/voice-interface.css">

// Create an instance
const voiceInterface = new VoiceInterface({
  chatbotId: 'your-chatbot-id',
  onResponse: (data) => {
    // Handle the response
    console.log(data);
  }
});

// Render the component
ReactDOM.render(
  voiceInterface,
  document.getElementById('voice-container')
);
