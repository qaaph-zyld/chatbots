Getting Started
Quick Start Guide

Follow these steps to start using the Chatbots Platform API:

  1. Create an account or sign in
  2. Generate an API key in your account settings
  3. Use the API key to authenticate your requests
  4. Create a chatbot using the API
  5. Customize your chatbot with personalities, knowledge bases, and plugins
  6. Integrate the chatbot into your application
Authentication
Authentication Methods

The API supports two authentication methods:

  • JWT Token: For user authentication and accessing user-specific resources
  • API Key: For application authentication and accessing chatbot resources

Include your JWT token in the Authorization header:

Authorization: Bearer your-jwt-token

Include your API key in the X-API-Key header:

X-API-Key: your-api-key
Rate Limiting
API Rate Limits

To ensure fair usage, the API has rate limits for different endpoints:

  • Authentication endpoints: 10-20 requests per minute
  • Chatbot conversation endpoints: 60 requests per minute
  • Management endpoints: 30 requests per minute

If you exceed the rate limit, you'll receive a 429 Too Many Requests response.
