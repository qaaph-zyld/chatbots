Build Powerful Chatbots with Freedom

A fully open-source platform for creating customizable chatbots with advanced AI capabilities, including voice interface - all without relying on proprietary services.

Key Features

Customizable Personalities

Create chatbots with distinct personalities, tones, and language styles to match your brand or use case.

Knowledge Base Integration

Connect your chatbots to custom knowledge bases using open-source vector databases.

Plugin Architecture

Extend functionality with plugins for sentiment analysis, data processing, and more.

Voice Interface

Add voice capabilities with local speech-to-text and text-to-speech processing.

Privacy-Focused

Process all data locally without sending sensitive information to external services.

Fully Open Source

Built entirely with open-source components, giving you complete freedom and control.

Advanced Voice Interface

Our platform includes a comprehensive open-source voice interface with local processing capabilities:

  • Local speech-to-text with Mozilla DeepSpeech and Kaldi
  • High-quality text-to-speech with Coqui TTS
  • Speaker identification and verification
  • Multi-language support
  • Voice activity detection and noise filtering
  • Optimized for various hardware configurations

Get Started in Minutes

# Install dependencies
cd chatbots
npm install

# Start the development server
npm run dev
